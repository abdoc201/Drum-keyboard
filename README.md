
# Drum Keyboard

Drum App is a simple and interactive web application that allows users to play various drum sounds by pressing corresponding keys on the keyboard. It provides an enjoyable way to create rhythmic beats and explore different sounds.


## Technologies Used

- HTML
- CSS
- JavaScript
## Usage

- Press the keys 'A', 'S', 'D', 'F', 'G', 'H', 'J', 'K', 'L' to play different drum sounds.
- Enjoy creating your own beats!
